# Learning: keep detail section headings visually consistent

## What changed

The `Evolution Chain` section on the Pokemon detail page now uses the same container rhythm as the other detail sections: `px-4 py-10 max-w-6xl mx-auto`, plus the same `Fredoka` heading treatment with the vertical accent bar.

## Why it matters

Even when the section content is unique, both the heading pattern and the section container should stay consistent so the detail page reads like one cohesive surface instead of a mix of unrelated section styles.
